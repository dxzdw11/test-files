Internet Speed Test Tool
A simple command-line utility written in Batch to quickly test your internet connection speed by downloading test files of various sizes.

Features
Quick Download Tests: Supports 1MB and 10MB test files.

Lightweight: No installation required; relies entirely on native Windows tools (curl).

Open Source: Managed via GitHub.

Requirements
Operating System: Windows 10 or later (with curl pre-installed).

Internet Connection: Required to perform the download tests.

How to use the script
Ensure the script.bat file is located in your project folder.

Run the file by double-clicking script.bat or by executing it via Terminal/PowerShell.

Follow the on-screen instructions:

Type 1 for the 1MB test.

Type 2 for the 10MB test.

The script will download the file and display the download statistics provided by curl.

Contributing
Contributions are welcome! If you want to add new features, such as automatic speed calculation in Mbps or support for larger files, feel free to open a Pull Request.
